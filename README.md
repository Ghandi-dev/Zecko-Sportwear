## Run Locally

Clone the project

```bash
  git clone https://github.com/Ghandi-dev/Zecko-Sportwear.git
```

Go to the project directory

```bash
  cd Zecko-Sportwear
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
